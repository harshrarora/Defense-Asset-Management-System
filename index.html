🔷

1NF

All attributes are atomic. No repeating groups. Each row is uniquely identifiable via primary keys.

🔶

2NF

No partial dependencies. All non-key attributes fully depend on the entire primary key.

🔹

3NF / BCNF

No transitive dependencies. Manufacturer details separated from weapon data. Every determinant is a candidate key.
