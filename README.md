# Defense-Asset-Management-System

A DBMS mini-project developed using Oracle SQL and PL/SQL for tracking defense assets, weapon inventory, maintenance history, ammunition compatibility, and custody management.

## Technologies
- Oracle SQL
- PL/SQL
- HTML/CSS/JavaScript
- Oracle Live SQL

## Features
- 10 normalized tables
- Stored procedures
- Functions
- Triggers
- Transactions
- Inventory dashboard
- Analytics visualization

## Project Structure

- /sql → Oracle SQL scripts
- /assets → Screenshots and diagrams
- /css, /js → Website frontend
- index.html → Main dashboard
- Final_Report.pdf → Project report

## How to Run

1. Open Oracle Live SQL
2. Run SQL files sequentially
3. Open index.html in browser
